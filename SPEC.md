# 云服务商前端页面规格

## 项目概述
- **项目名称**: CloudServerManager 云服务商主页
- **类型**: Vue 3 单页面前端
- **核心功能**: 展示云服务器套餐，提供导航与联系入口
- **目标用户**: 需要购买云服务器或托管服务的客户

## 视觉规格

### 色彩系统
| 用途 | 色值 |
|------|------|
| 主色（背景） | #FEDFE1（樱花粉） |
| 次级背景 | #FFF8F9 |
| 强调色 | #E8A0B0（深樱粉） |
| 文字主色 | #4A4A4A |
| 文字次色 | #7A7A7A |
| 卡片背景 | #FFFFFF |
| 按钮主色 | #D4738A |
| 按钮悬停 | #C45C75 |

### 字体
- 主字体: "Noto Sans SC", sans-serif
- 标题: 600 weight
- 正文: 400 weight

### 布局
- 最大宽度: 1200px，居中
- 导航栏: 固定顶部，白色背景带阴影
- 内容区: 响应式，桌面端3列套餐卡片，移动端单列

## 页面结构

### 导航栏
- Logo: "CloudServerManager" 左侧
- 导航按钮（右侧）: 主页 | 云服务器 | 机房托管 | 联系我们 | 控制台
- 移动端: 汉堡菜单

### Hero 区域
- 主标题: "畅享稳定高效的云端体验"
- 副标题: "企业级云服务器，即开即用"

## 交互规格

### 导航
- 点击"控制台"→ 新窗口打开 # (示例链接)
- 点击"联系我们"→ 滚动至页脚联系方式
- 其他导航按钮→ 当前页展示（占位）

### 套餐卡片
- 悬停: 轻微上浮 + 阴影加深
- "立即购买"按钮: 悬停变色

### 移动端
- 断点: 768px
- 移动端导航折叠为汉堡菜单
- 套餐卡片单列展示

## 技术栈
- Vue 3 (CDN引入)
- 单HTML文件，内嵌Vue组件
- CSS3（Flexbox + Grid）
- 无需构建工具
