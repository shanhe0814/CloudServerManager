# CloudServe 云服务商前端页面需求

## 需求概要

用户需要基于 Vue.js 创建一个云服务商（CloudServe）的前端页面。

## 详细需求

| 项目 | 要求 |
|------|------|
| 主色调 | 樱花色 #FEDFE1 |
| 界面风格 | 简洁，不冗杂，便于用户判断 |
| 服务器套餐 | 3个热销套餐：¥29/¥79/¥199 每月（展示在主页） |
| 导航栏按钮 | 主页、云服务器、机房托管、联系我们、控制台 |

- 入门型：¥29/月（配置待定）
- 标准型：¥79/月（配置待定）
- 旗舰型：¥199/月（配置待定）

## 部署方式

- **Docker部署**：使用 `docker-compose up -d` 启动
- **容器**：nginx:alpine，端口8080:80
- **启动命令**：`docker-compose up -d --build`
- **停止命令**：`docker docker-compose down`
